# ✨ To-Do List App

A clean and animated To-Do List built with React.js

## Features
- Add new tasks
- Delete tasks
- Move tasks up and down
- Animated UI with colorful design
- Tasks saved in localStorage (persist on refresh)

## Tech Stack
- React.js
- CSS3 Animations
- localStorage

## How to Run
```bash
npm install
npm start
```
